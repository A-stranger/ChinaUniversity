# 全国高等学校名单

最后更新日期：2017 年 6 月 14 日

## 更新规则

1. 每年高考结束后，根据教育部发布的最新《全国高等学校名单》进行更新，数据截止至当年 5 月 31 日。

2. 港、澳、台大学列表，不定时进行修订。
